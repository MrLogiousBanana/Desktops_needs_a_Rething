# -ENG-Desktop-in-the-Age-of-Background-Apps-Why-DE-Needs-a-Rethink
The essay explains why modern Windows/macOS/Linux desktops struggle with background apps: fragmented notifications, tray clutter, and blurred lines between apps and services. It proposes OS‑level app classes and permissioned access to tray, autostart, and advanced notifications to restore consistency and user control.
